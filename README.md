# TubesAKA

**Usulan Topik/Judul**
> Analisis Kompleksitas Algoritma Iteratif dan Rekursif pada Proses Pengurutan Data Rating Produk Menggunakan Selection Sort dan Quick Sort

**Deskripsi**
> Dalam sistem yang menampilkan data produk berdasarkan rating atau penilaian pengguna, proses pengurutan (sorting) berperan penting untuk menyajikan informasi dengan cepat dan akurat. Pada studi kasus ini digunakan dua algoritma, yaitu Selection Sort yang menggunakan pendekatan iteratif, dan Quick Sort yang menggunakan pendekatan rekursif. Keduanya akan dibandingkan dalam hal efisiensi waktu eksekusi dan kompleksitas waktu pada berbagai ukuran data rating produk. Analisis ini bertujuan untuk menentukan algoritma yang paling optimal dalam konteks pengurutan data berbasis rating pengguna.
